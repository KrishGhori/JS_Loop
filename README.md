# JavaScript Mini Programs

This project contains multiple **basic JavaScript programs** created using **HTML, CSS, and JavaScript**.  
It is designed to practice **conditional statements and loops** in JavaScript.

##  Pages Included

1. **Positive Number Checker**
   - Enter a number and check whether it is **positive, negative, or zero**.

2. **Voting Eligibility**
   - Checks whether a person is **eligible to vote (age ≥ 18)**.

3. **Percentage Calculator**
   - Calculates percentage based on entered marks.

4. **Calculator**
   - Performs basic operations like **addition, subtraction, multiplication, and division**.

5. **For Loop Example**
   - Demonstrates how a **for loop** works in JavaScript.

6. **While Loop Example**
   - Demonstrates how a **while loop** works.

7. **Do While Loop Example**
   - Demonstrates **do...while loop execution**.

8. **For In Loop Example**
   - Demonstrates how **for...in loop** works in JavaScript objects.

## 🛠 Technologies Used

- HTML5

- JavaScript  

##  Features

- Simple and clean user interface
- Interactive JavaScript logic
- Easy navigation using a navbar
- Beginner-friendly examples
