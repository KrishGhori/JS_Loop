# JS_Loop

## JavaScript Mini Programs

This project contains several **basic JavaScript programs** created using **HTML, CSS, and JavaScript**.
The purpose of this project is to practice **JavaScript fundamentals**, including **conditional statements and loop structures**.

---

## Pages Included

### 1. Positive Number Checker

* Allows the user to enter a number.
* Determines whether the number is **positive, negative, or zero**.

### 2. Voting Eligibility

* Accepts a user's age as input.
* Checks whether the person is **eligible to vote (age ≥ 18)**.

### 3. Percentage Calculator

* Calculates the **percentage based on entered marks**.

### 4. Calculator

* Performs basic arithmetic operations:

  * Addition
  * Subtraction
  * Multiplication
  * Division
  * Modulus

### 5. For Loop Example

* Demonstrates how a **for loop** works by printing patterns or sequences.

### 6. While Loop Example

* Demonstrates how a **while loop** executes based on a condition.

### 7. Do While Loop Example

* Shows how a **do...while loop** runs at least once before checking the condition.

### 8. For In Loop Example

* Demonstrates the **for...in loop** used to iterate over properties of JavaScript objects.
